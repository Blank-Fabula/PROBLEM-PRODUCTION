# 제목: 타일 교체

## 문제 링크
### 　[**[링크(AScode)]**]()

## 문제 설명
가장 많이 존재하는 타일로 나머지 모든 타일을 교체하자.
　
## 입력 설명
int 크기의 양수 M, N이 입력된다. N은 M의 약수이며, N * N 크기의 타일들로 이루어진 M * M 크기의 공간이 입력된다.

## 출력 설명
가장 많이 존재하는 타일로 M * M 공간을 채운 모양을 출력한다. (가장 많이 존재하는 타일이 여러개의 경우 사전순으로 빠른 것으로 채운다.)

XCBS
